# intention
- I've started learning guitar using the "Absolutely Understand Guitar" method from Scotty West
- Scotty's advice to learn faster is to train:
  - for 5 minutes
  - with drums (using a drum machine)
  - on a slow enough tempo
  - alternating between two random chords

This project aim at providing on a webpage:
- a simple drum machine where I can choose between simple 4/4 rock type beats from a selection
- tempo selectoon
- pick two chords randomly (start between major, minor, 7th, minor 7th, and powerchords) (with a random option)
- pick the finger shape for the chord (E, A, G, ...) (with a random option)
- see the name and fingering for the chords
- optionally adds a tuner
- a timer for 5 minutes exercices
- a streak monitor for each day i spend doing the exercice (and how many time per day)
